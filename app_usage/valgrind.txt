#+TITILE: Valgrind - Dynamic analysis to detect memory and threading bugs

 - [[http://valgrind.org/][valgrind website]]
 - Valgrind is an instrumentation framework for building dynamic analysis tools.
   There are Valgrind tools that can automatically detect many memory management
   and threading bugs, and profile your programs in detail. You can also use Valgrind
   to build new tools.

 - usage
   - valgrind leak-check=yes [your program]
